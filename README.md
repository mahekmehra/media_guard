# 🛡️ Media Guard (Emotion-Safe AI)

Media Guard is an AI system designed to detect emotionally manipulative content in news and social media, explain why such content can be harmful, and rewrite it into a neutral, mentally safe version using ML and Generative AI.

## Current Phase
Phase 0 – Project foundation with backend API and frontend UI.

## Tech Stack
- Backend: FastAPI
- Frontend: Streamlit
- Language: Python

## Roadmap
- Phase 1: Emotion detection using NLP models  
- Phase 2: Manipulation risk scoring  
- Phase 3: GenAI explanation  
- Phase 4: Neutral rewrite  
- Phase 5: RAG-based news research assistant
